Movie Recommendation System
← Back to Projects Page
Introduction
This is a Machine Learning application that recommends similar movies based on the given input movie.
Data used:
TMDB Movies Dataset Click Here
The dataset contained lots of imformation about movies of various languages and regions. The data was not ready for model training and required lots of data processing and feature engineering steps.
Workflow:
  • Data preperation
  • Removing inappropriate Data
  • Filling missing values
  • Handling Outliers
  • Encoding Features
  • Selecting important Features
  • Converted them to Vectors using Word Embeddings
  • Model Training
  • Calculated cosine similarities between vectors
  • Defined a function to predict 5 similar vectors(movies)
  • Deployment
  • Created web page using HTML,CSS
  • Integrated model with web page using Flask
  • Uploaded to GitHub
Libraries and Frameworks used:
Python
Pandas
Numpy
Matplotlib
Sklearn
Ginsim
HTML
CSS
Flask
Conclusion:
By doing this project I gained some experience in Machine Learing and NLP and got knowledge to build a full Machine Learing model

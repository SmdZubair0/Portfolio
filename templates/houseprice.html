Introduction
This is a Machine Learning application to predict prices of different houses present in Bangalore city. I made this project in order to showcase my Machine Learning skills.
Data used:
Banglore House Price Dataset Click Here
The dataset contained lots of imformation about different houses in bangalore. The data was not ready for model training and required lots of data processing and feature engineering steps.
Workflow:
  • Data preperation
  • Removing inappropriate Data
  • Filling missing values
  • Handling Outliers
  • Encoding Features
  • Selecting important Features
  • Model Training and Evaluation
  • Splited data into training and testing sets
  • Trained different models like LinearRegression, DecisionTreeRegressor, Ensemble models etc.
  • Performed Hyperparameter tuning
  • Evaluated performance using metrics like r2 score
  • Finally Selected StackingRegressor model
  • Deployment
  • Created web page using HTML,CSS
  • Integrated model with web page using Flask
  • Uploaded to GitHub

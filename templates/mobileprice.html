Introduction
This is a Machine Learning application to predict prices of different mobiles as per Flipkart. I made this project in order to showcase my Machine Learning skills.
Data used:
Mobile Price Dataset Click Here
I created this dataset by collecting data of various mobile phones through web scrapping. The data collected was raw. I had to perform a lots of preprocessing to make it usable
Workflow:
  • Data cleaning
  • Data preperation
  • Removing inappropriate Data
  • Filling missing values
  • Handling Outliers
  • Encoding Features
  • Selecting important Features
  • Model Training and Evaluation
  • Splited data into training and testing sets
  • Trained different models like LinearRegression, DecisionTreeRegressor, Ensemble models etc.
  • Performed Hyperparameter tuning
  • Evaluated performance using metrics like r2 score
  • Finally Selected RandomForestRegressor model
  • Deployment
  • Created web page using HTML,CSS
  • Integrated model with web page using Flask
  • Uploaded to GitHub
